## what
- [[IPアドレス|サブネット]]内の通信が「次にどこへ向かうべきか」を決める「行き先リスト」
- 送信先 (Destination)とターゲット (Target)のリストで構成される
	- Destination: どこへ行きたいか
	- Target: どこを経由するか

| Destination   | Target  |                         |
| ------------- | ------- | ----------------------- |
| `10.0.0.0/16` | local   | VPC内の通信なら、local(VPC内)へ  |
| 0.0.0.0/0     | igw-xxx | インターネット宛はインターネットゲートウェイへ |
- サブネットに紐づけることで利用可能になる
