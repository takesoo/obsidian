- できること
	- エクセル上のデータを集計する


- 設定方法
	- Microsoft 365 Insider Programに参加し、ベータチャンネルInsiderレベルを選択する
	- リボンで\[数式\]を選択
	- \[Pythonの挿入\]を選択
	- \[プレビューで試す\]を選択

- 使用できる環境
	- Excel for Windowsのみ

- Pythonはクラウドで実行される
