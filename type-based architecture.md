## what
- [[React]]および[[Next.js]]でのディレクトリ構成戦略のひとつ
- 「技術的な種類」に基づいて分ける
```
src/
├── api/                # APIフック、API定義
├── assets/             # 画像やフォントなど
├── components/         # UIコンポーネント
├── contexts/           # React Context
├── hooks/              # カスタムフック
├── stores/             # グローバルステート
├── styles/             # CSS
├── types/              # 型定義
├── utils/              # ユーティリティ関数
...
```
