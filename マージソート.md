配列を最小単位まで分割し、整列させながら元に戻す方法。
1. 配列を最小単位まで分割する
	\[4, 3, 9, 8, 2, 7, 5, 1]
	\[4, 3, 9, 8], \[2, 7, 5, 1]
	\[4, 3], \[9, 8], \[2, 7], \[5, 1]
2. それぞれソートする
	\[3, 4], \[8, 9], \[2, 7], \[1, 5]
3. ソートしながらマージする
	\[3, 4, 8, 9], \[1, 2, 5, 7]
4. 3を繰り返す
	\[1, 2, 3, 4, 5, 7, 8, 9]

O(n * log n), Ω(n * log n)
