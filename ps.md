- プロセスの確認
- 表示される項目
	- PID
	- TTY
	- STAT
	- TIME
	- COMMAND

- オプション
	- a:  全てのプロセス
	- u: ユーザー名を表示
	- x: [[デーモンプロセス|デーモン]]も表示

```bash
p [options]

ps
  PID TTY          TIME CMD
   12 pts/1    00:00:00 bash
22869 pts/1    00:00:00 ps

# a すべてのプロセスを表示する
# f プロセスの親子関係を表示する
# u 実行ユーザーを表示する
# x 制御端末のないバックグラウンドプロセスも表示する
```
