## what
- 認証認可機能を提供するサービス
- ユーザープール：認証機能を提供する
	- ユーザーの登録、サインイン、パスワードリセット
	- ユーザープロファイルの管理
	- MFA
	- 外部IDプロバイダーとの連携
	- JWTを発行する
		- [[AWS API Gateway]]のCognitoオーソライザーと連携し、JWTを持つユーザーのみAPIの実行を許可する
- IDプール：認可機能を提供する
	- 認証されたユーザーに対して、他のAWSサービスにアクセスするための一時的なIAMロールを付与する
		- [[AWS STS]]を通じて、一時的なアクセスキー、シークレットキー、セッショントークンを発行する
