## プログラムの4つの性質
|                    |                                                                                              |
| ------------------ | -------------------------------------------------------------------------------------------- |
| [[リロケータブル]] | プログラムを主記憶上のどこに配置しても実行可能であること                                     |
| [[リユーザブル]]   | 主記憶上に読み込まれたプログラムは、それ以降は再ロードしなくても繰り返し利用可能であること。 |
| [[リエントラント]] | 複数のタスクからの呼び出しに対して並行して実行可能であること。                               |
| [[リカーシブ]]     |    プログラム実行中にそのプログラム自分自身を呼び出すことができること。                                                                                          |
