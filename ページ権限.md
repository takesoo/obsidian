- ページごとに設定できる権限

|                        | Full Access | Can Edit | Can Comment | Can Read | Disabled |
| ---------------------- | ----------- | -------- | ----------- | -------- | -------- |
| 別ユーザーへの権限付与 | o           | x        | x           | x        | x        |
| 対象ページの編集・削除 | o           | o        | x           | x        | x        |
| 対象ページへのコメント | o           | o        | o           | x        | x        |
| 対象ページの閲覧       | o           | o        | o           | o        | x         |

- 権限の付与はページ右上のShareから
- 親のページの権限を継承する
