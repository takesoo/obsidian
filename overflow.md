## what
- コンテンツが要素のパディングボックスからはみ出す時の水平方向と垂直方向の動作設定

| 値         | コンテンツ                               | スクロールバー          |
| --------- | ----------------------------------- | ---------------- |
| `visible` | パディングボックスからはみ出す                     | x                |
| `hidden`  | パディングボックスで切り取り                      | x                |
| `clip`    | overflow-clip-merginで定義したクリップ辺で切り取り | x                |
| `scroll`  | パディングボックスで切り取り                      | 常に表示             |
| `auto`    | パディングボックスで切り取り                      | overflowしたときだけ表示 |
