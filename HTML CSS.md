## display要素

|              | 特徴                                                 | 要素                                                                                              |
| ------------ | -------------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| block        | - 親要素の横幅いっぱいに広がる<br>- 縦に配置されていく<br>- 自由な高さと幅を指定できる | - `<h1>`~`<h6>`<br>- `<p>`<br>- `<ul>`<br>- `<div>`<br>- `<header>`,`<footer>`<br>- `<section>` |
| inline       | - 高さを指定できない<br>- 横に配置されていき、限界点で折り返す                | - `<span>`<br>- `<img>`<br>- `<a>`<br>- `<input>`<br>- `<textarea>`                             |
| inline-block | - インライン要素のように横に配置され、ブロック要素のように幅や高さを指定できる。          | displayプロパティで指定する                                                                               |


## 相対単位
- `em`: 親要素の`font-size`を1とした単位。
- `rem`: ルート要素の`font-size`を1とした単位
- `%`: 本来の長さ(`width`, `height`)を100とした単位
- `vh`, `vw`: [[viewport]]の高さ、幅を100とした単位
- `svh`, `svw`: アドレスバーなどが表示された状態の最小[[viewport]]の高さ、幅を100とした単位
- `lvh`, `lvw`: アドレスバーなどが非表示になった状態の最大[[viewport]]の高さ、幅を100とした単位
- `dvh`, `dvw`: アドレスバーなどの表示・非表示に合わせて動的に[[viewport]]の高さ、幅が変化する

## Overflow
- ブロック要素はデフォルトでコンテンツに合わせてサイズが変わる
- widthとheightを固定した場合、コンテンツの量が多いとはみ出す→Overflow
- `overflow`プロパティで隠したりスクロールにしたり
- `text-overflow`プロパティでコンテンツを切る
