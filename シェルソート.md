データを一定間隔ごとに取り出し、そのデータ内で並び替えて元に戻す。次に、間隔を狭めて取り出し、並べ替える。これを間隔が１になるまで繰り返す。
1. 4個間隔で値を取り出す
	\[==8==, 5, 7, 9, ==2==, 0, 3, 4] => \[8, 2], \[5, 0], \[7, 3], \[9, 4]
2. それぞれを並べ替える
	\[2, 8], \[0, 5], \[3, 7], \[4, 9]
3. 元の配列に戻す
	\[==2==, 0, ==3==, 4, ==8==, 5, ==7==, 9]
4. 2個間隔で1, 2, 3を繰り返す
	\[2, 3, 8, 7], \[0, 4, 5, 9]
	\[2, 3, 7, 8], \[0, 4, 5, 9]
	\[2, 0, 3, 4, 7, 5, 8, 9]
	\[0, 2, 3, 4, 5, 7, 8, 9]
