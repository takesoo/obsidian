- CUI環境においてテキストファイルを１ページごとに区切って表示すること
- [[ページャ]]で呼び出される画面

## 操作方法

| キー   | 動作                             |
| ---- | ------------------------------ |
| スペース | 次のページに進む                       |
| b    | 前の一画面に戻る(back)                 |
| f    | 次の一画面に進む(forward)              |
| ↑    | 前の行に戻る                         |
| ↓    | 次の行に進む                         |
| \/単語 | 単語を検索する。n(next)で次の検索結果にジャンプする。 |
| q    | [[ページャ]]コマンドを終了(quit)する        |
