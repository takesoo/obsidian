# RDSとは

フルマネージドなDBサーバー

バックアップ、ソフトウェアアップデートが自動でされる

マルチAZが推奨されている

設定には設定グループを作成して割り当てる形式で行う

## DBパラメータグループ

DB設定値の設定（接続数など）

## DBオプショングループ

追加機能設定

## DBサブネットグループ

RDSを設置するサブネットの設定

# 構築方法

## 1. セキュリティグループ作成

インバウンドルールで3306をwebサーバーのセキュリティグループ(aws-and-infra-web)に開放

webサーバーを冗長化した際に個別にipを登録するのは難しい、そこでSGに属するサーバーをまとめて設定できるようになっている
