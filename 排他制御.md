1つの[[トランザクション処理]]を行なっている間はデータベースにロックをかけて他のトランザクションの影響を受けないようにすること。
[[ACID特性]]を担保するための1つの手段。

## 専有ロックと共有ロック
|            |                                                    |
| ---------- | -------------------------------------------------- |
| 共有ロック | 他のユーザーはデータを参照できるが更新はできない。 |
| 専有ロック | 他のユーザーはデータの参照も更新もできない。                                                   |
