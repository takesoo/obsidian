

| **形式** | **特徴**                       | **メリット**                                        | **デメリット**                        |
| ------ | ---------------------------- | ----------------------------------------------- | -------------------------------- |
| MP3    | 圧縮形式。広く普及しており、ほとんどのデバイスで再生可能 | ・ファイルサイズが小さい  <br>・汎用性が高い                       | ・音質が劣化する  <br>・高音質を求める場合には不向き    |
| WAV    | 非圧縮形式。CDと同じ音質を保持             | ・音質が非常に高い  <br>・編集に適している                        | ・ファイルサイズが非常に大きい  <br>・ストレージを多く消費 |
| FLAC   | 可逆圧縮形式。音質を保ちながらファイルサイズを圧縮    | ・高音質を維持  <br>・ファイルサイズがWAVより小さい  <br>・メタデータ管理が容易 | ・一部のデバイスで再生できない場合がある             |
| AAC    | 圧縮形式。MP3よりも高音質で効率的           | ・MP3よりも音質が良い  <br>・ファイルサイズが小さい                  | ・一部の古いデバイスでは非対応                  |
| ALAC   | Appleの可逆圧縮形式。FLACと同様の音質      | ・高音質のまま  <br>・Apple製品との互換性が高い                   | ・Apple製品以外では互換性が低い               |

| **ソフト名**                   | **料金**    | **対応OS**              | **主な特徴**                                 | **対応ファイル形式 (代表例)**               | **操作の難易度**           |
| -------------------------- | --------- | --------------------- | ---------------------------------------- | -------------------------------- | -------------------- |
| **Exact Audio Copy (EAC)** | 無料        | Windows               | **究極の正確性**。エラー補正とAccurateRipにより完璧な複製が可能。 | WAV, FLAC, MP3, WMA, etc.        | **高** (初期設定が複雑)      |
| **dBpoweramp**             | 有料 (試用あり) | Windows / macOS       | **高速・高機能**。マルチコア対応で速く、メタデータ取得が非常に強力。     | FLAC, ALAC, MP3, AAC, AIFF, etc. | **低〜中** (直感的で使いやすい)  |
| **MusicBee**               | 無料        | Windows               | **管理機能の充実**。リッピングからタグ編集、鑑賞まで1つで完結。       | FLAC, MP3, AAC, OGG, Opus, etc.  | **中** (多機能だが整理されている) |
| **foobar2000**             | 無料        | Windows / (macOS版もあり) | **超軽量・拡張性**。好みのコンポーネントを追加して機能をカスタマイズ。    | ほぼ全ての形式に対応 (要プラグイン)              | **高** (自分好みに育てる必要あり) |
| **Apple Music / iTunes**   | 無料        | Windows / macOS       | **Apple製品との親和性**。iPhone等への同期がスムーズで最も手軽。  | ALAC, AAC, MP3, AIFF, WAV        | **低** (初心者でも迷わない)    |
